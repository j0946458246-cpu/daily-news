# 蚂蚁灵波发布LingBot-VA 2.0动作模型

**2026年7月17日 12时34分56秒**

---

蚂蚁灵波发布具身原生世界动作模型LingBot-VA 2.0，定位为“视觉-动作”端到端策略，支持机械臂抓取、移动操作与多物体排序。与语言模型不同，VA模型直接在像素/点云到关节扭矩间建立映射，训练依赖海量仿真（Isaac Gym/Mujoco）与真实遥操作数据。2.0版本重点改进了“泛化到未知物体”与“人机协作避障”。蚂蚁入局具身智能，逻辑是用支付、零售、物流场景养数据：仓内分拣、商户巡检、无人收银台都是天然试验场。金融公司做机器人看似跨界，实则是在为“线下商业自动化”储备技术资产。
