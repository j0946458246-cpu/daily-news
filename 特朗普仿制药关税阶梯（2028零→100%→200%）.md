# 特朗普仿制药关税阶梯（2028 零→100%→200%）

**2026年7月17日 12时34分56秒**

---

7 月 21 日特朗普宣布 2026.8.1 起仿制药零关税两年，2028 起 100% 一年，后 200%。前批未写。印度仿制药占美近 40% 销量、对美出口 97 亿美，分析师称印制造成本低 40-60% 即使加税仍存优势，但中小 API 厂会被洗。对中国 CDMO 是双刃：短期订单避险转移，长期美本土建厂压力传导。A 股仿制药 7.22 异动（博瑞+9%）。政策若落地，2028 后全球仿制药版图重划，中国原料药需转"美国本土合资+中国中间体"模式。
